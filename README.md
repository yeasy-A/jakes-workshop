# Jake's Workshop

A small collection of free, single-file web tools. No build step, no backend —
each app is one self-contained page you can open in a browser.

## Apps

- **VA Toolkit** (`va/`) — disability-rating calculator, claim helper, PACT Act and benefits references, cited to VA.gov.
- **Veteran Finder** (`veteran-finder/`) — resource lookup for veterans, with crisis-line info always in reach.
- **Sparks** (`sparks/`) — five quick brain-training games.
- **PatternMap** (`patternmap/`) — a navigable graph of recurring biblical patterns.
- **Tongues** (`tongues/`) — a language-practice tool.
- **Billet** (`jobs/`) — a job-application helper (showcase build).
- **Faith** (`faith/`) — prayers and daily readings.

## Hosting

Static site — deploys to GitHub Pages or Cloudflare Pages as-is. No build command;
the repository root is the site root.
